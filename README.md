## рџ¤– What It Does

**StockSense AI** predicts whether a stock will move **+1.5% or more** in the next **5 trading days**.

### Input
- 60+ OHLCV candles (date, open, high, low, close, volume)

### Output
- `BUY` or `HOLD/SELL` signal
- Confidence score (0вЂ“100%)
- SHAP explanation: which indicators drove this prediction
- Inference time

### Features (27 Technical Indicators)
| Category    | Indicators                                      |
|-------------|--------------------------------------------------|
| Trend       | EMA 9/21 cross, SMA 50, Price/SMA50             |
| Momentum    | RSI 14, MACD, ROC 5/10, Momentum 10/20         |
| Volatility  | Bollinger %B & Width, ATR 14, Vol 10/20d        |
| Volume      | Volume change, Volume/MA ratio                  |
| Lag Returns | 1d, 2d, 3d, 5d, 10d returns                    |
